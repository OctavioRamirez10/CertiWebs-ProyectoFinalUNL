💡 Conceptos Clave

Direcciones IP y Subnetting

# Ejemplo de configuración IP IP: 192.168.1.10 Máscara: 255.255.255.0 (/24) Gateway: 192.168.1.1 DNS: 8.8.8.8 # Cálculo de subredes Red: 192.168.1.0/24 Hosts disponibles: 254 Primera IP: 192.168.1.1 Última IP: 192.168.1.254 Broadcast: 192.168.1.255

Comandos de Red Básicos

# Windows ipconfig /all # Muestra configuración IP ping google.com # Prueba de conectividad tracert google.com # Ruta hacia destino netstat -an # Conexiones activas nslookup google.com # Resolución DNS # Linux/Mac ifconfig -a # Muestra interfaces ping -c 4 google.com # 4 paquetes ICMP traceroute google.com # Ruta hacia destino netstat -tuln # Puertos abiertos dig google.com # Consulta DNS detallada

Protocolos Comunes

# Puertos TCP/UDP estándar HTTP: 80/tcp # Navegación web HTTPS: 443/tcp # Navegación segura FTP: 21/tcp # Transferencia archivos SSH: 22/tcp # Conexión remota Telnet: 23/tcp # Terminal remoto SMTP: 25/tcp # Envío email DNS: 53/udp # Resolución nombres DHCP: 67/udp # Asignación IP

Modelo OSI vs TCP/IP

OSI (7 capas) TCP/IP (4 capas) 7. Aplicación 4. Aplicación 6. Presentación 3. Transporte 5. Sesión 2. Internet 4. Transporte 1. Acceso Red 3. Red 2. Enlace 1. Físico
