Sistemas Operativos - Guía de Estudio

Aprende los fundamentos de sistemas operativos Windows y Linux

¿Qué es un Sistema Operativo?

Un Sistema Operativo (SO) es el software fundamental que gestiona los recursos de hardware y proporciona servicios a los programas de aplicación. Actúa como intermediario entre el usuario y el hardware de la computadora.

Funciones Principales

  • Gestión de procesos: Creación, ejecución y terminación de programas
  • Gestión de memoria: Asignación y liberación de memoria
  • Gestión de archivos: Organización y acceso a datos
  • Gestión de dispositivos: Control de hardware y periféricos
  • Interfaz de usuario: Comunicación con el usuario
  • Seguridad: Protección y control de acceso

Tipos de Sistemas Operativos

  • Monousuario: Un usuario a la vez (Windows Home)
  • Multiusuario: Múltiples usuarios simultáneos (Linux, Unix)
  • Monotarea: Un proceso a la vez
  • Multitarea: Múltiples procesos simultáneos
  • De tiempo real: Respuesta inmediata
  • Distribuido: Múltiples computadoras conectadas

Windows

Características de Windows

  • Interfaz gráfica: GUI intuitiva y fácil de usar
  • Compatibilidad: Amplio soporte de software y hardware
  • Active Directory: Gestión centralizada de usuarios
  • PowerShell: Interfaz de línea de comandos avanzada
  • Windows Update: Sistema de actualizaciones automático
  • Seguridad: Windows Defender, Firewall, BitLocker

Comandos Básicos de Windows

Get-Process - Listar procesos

Get-Service - Listar servicios
Test-Connection - Probar conectividad

Gestión de Archivos y Permisos

  • NTFS: Sistema de archivos con seguridad
  • ACLs: Listas de control de acceso
  • Permisos: Lectura, escritura, ejecución, modificación
  • Herencia: Propagación de permisos
  • Propietario: Control de archivos

Linux

Características de Linux

  • Open Source: Código fuente abierto y gratuito
  • Multiusuario: Múltiples usuarios simultáneos
  • Seguridad: Modelo de permisos robusto
  • Personalización: Altamente configurable
  • Estabilidad: Sistema muy estable y confiable
  • Distribuciones: Ubuntu, CentOS, Debian, Fedora, Arch

Comandos Básicos de Linux

# Navegación de archivos
ls - Listar archivos
cd - Cambiar directorio

mkdir - Crear directorio
rmdir - Eliminar directorio vacío
rm - Eliminar archivos/directorios
cp - Copiar archivos

# Gestión de archivos

grep - Buscar texto en archivos
find - Buscar archivos
chmod - Cambiar permisos
chown - Cambiar propietario

# Sistema

kill - Terminar proceso

free - Memoria disponible
uname - Información del sistema
whoami - Usuario actual
sudo - Ejecutar como administrador

Estructura de Directorios Linux

  • / - Root: Directorio raíz
  • /home: Directorios de usuarios
  • /bin: Comandos esenciales
  • /etc: Configuración del sistema
  • /var: Datos variables (logs, caché)
  • /usr: Programas de usuario
  • /tmp: Archivos temporales
  • /opt: Software opcional

Gestión de Procesos

Conceptos de Procesos

  • Proceso: Programa en ejecución
  • PID: Identificador único de proceso
  • Hilos: Subprocesos dentro de un proceso
  • Estados: Nuevo, listo, ejecutando, bloqueado, terminado
  • Prioridades: Nivel de importancia del proceso

Gestión en Windows

# Administrador de tareas (GUI)
taskmgr.exe

# Línea de comandos
tasklist - Listar procesos
tasklist /v - Información detallada
taskkill /f /im proceso.exe - Forzar cierre
wmic process get name,processid - Lista con PID

# PowerShell
Get-Process - Listar procesos
Stop-Process -Name proceso - Terminar proceso
Start-Process - Iniciar proceso

Gestión en Linux

# Listar procesos
ps aux - Todos los procesos
ps -ef - Formato extendido
top - Monitor en tiempo real
htop - Monitor mejorado

# Controlar procesos
kill PID - Terminar proceso
killall nombre - Terminar por nombre
kill -9 PID - Forzar terminación
nice -n 10 comando - Ejecutar con baja prioridad
renice -n 5 PID - Cambiar prioridad

# Procesos en background
comando & - Ejecutar en background
jobs - Listar trabajos
fg %1 - Traer al foreground
bg %1 - Enviar al background

Gestión de Memoria

Conceptos de Memoria

  • Memoria RAM: Memoria principal volátil
  • Memoria Virtual: Extensión en disco
  • Paginación: Intercambio de páginas
  • Segmentación: División lógica de memoria
  • Cache: Memoria intermedia rápida

Windows Memory Management

# Ver uso de memoria
wmic OS get TotalVisibleMemorySize,FreePhysicalMemory

# Archivo de paginación
systempropertiesadvanced.exe - Configuración virtual

# Monitor de rendimiento
perfmon.msc - Performance Monitor

# PowerShell
Get-Counter -Counter "\Memory\Available MBytes"
Get-Process | Sort-Object WorkingSet -Descending | Select-Object -First 10

Linux Memory Management

# Ver uso de memoria
free -h - Memoria en formato humano
cat /proc/meminfo - Información detallada
vmstat - Estadísticas virtuales

# Swap (memoria virtual)
swapon -s - Ver uso de swap
swapon /archivo - Activar swap
swapoff /archivo - Desactivar swap

# Limpiar caché
echo 3 > /proc/sys/vm/drop_caches
sync - Sincronizar disco

Sistema de Archivos

Sistemas de Archivos Comunes

  • Windows: NTFS, FAT32, exFAT
  • Linux: ext4, ext3, ext2, XFS, Btrfs
  • Unix: UFS, ZFS
  • Mac: HFS+, APFS

Permisos en Linux

# Estructura de permisos
# rwx rwx rwx
# dueño grupo otros

# Ver permisos
ls -la - Listar con permisos
stat archivo - Información detallada

# Cambiar permisos
chmod 755 archivo - rwxr-xr-x
chmod u+x archivo - Añadir ejecutable al dueño
chmod g-w archivo - Quitar escritura al grupo
chmod o=r archivo - Solo lectura para otros

# Cambiar propietario
chown usuario:grupo archivo
chown -R usuario directorio - Recursivo

# Permisos especiales
chmod +s archivo - SUID/SGID
chmod +t directorio - Sticky bit

Permisos en Windows

# GUI - Propiedades del archivo
# Pestaña Seguridad -> Avanzado

# PowerShell
Get-Acl archivo - Ver ACL
Get-Permission archivo - Ver permisos

# CMD
icacls archivo - Listar permisos
icacls archivo /grant Usuario:F - Conceder control total
icacls archivo /deny Usuario:W - Denegar escritura
takeown /f archivo - Tomar propiedad

Redes y Conectividad

Configuración de Red en Windows

# Ver configuración
ipconfig /all - Configuración completa
ipconfig /release - Liberar IP
ipconfig /renew - Renovar IP

# Configuración avanzada
netsh interface ip show config
netsh interface ip set address "Ethernet" static 192.168.1.100 255.255.255.0 192.168.1.1

# Firewall
netsh advfirewall show allprofiles
netsh advfirewall set allprofiles state off

# Conexiones remotas
mstsc.exe - Escritorio remoto
ping servidor - Probar conectividad
tracert servidor - Ruta de paquetes

Configuración de Red en Linux

# Ver configuración
ip addr show - Interfaces y direcciones
ip route show - Tabla de rutas
ss -tuln - Puertos abiertos

# Configurar interfaz
ip addr add 192.168.1.100/24 dev eth0
ip link set eth0 up - Activar interfaz
ip route add default via 192.168.1.1

# Configuración persistente
/etc/network/interfaces - Debian/Ubuntu
/etc/sysconfig/network-scripts/ - RHEL/CentOS

# Herramientas de red
ping -c 4 servidor - Probar conectividad
traceroute servidor - Ruta de paquetes
netstat -tuln - Conexiones activas
ss -tuln - Alternativa moderna

# SSH - Acceso remoto
ssh usuario@servidor - Conectar
scp archivo usuario@servidor:/ruta - Copiar archivo

Plan de Estudio Recomendado

Fase 1: Fundamentos (2-3 semanas)

  • Conceptos básicos de sistemas operativos
  • Componentes y arquitectura
  • Tipos de SO y características
  • Historia y evolución
  • Terminología básica

Fase 2: Windows (3-4 semanas)

  • Instalación y configuración
  • Interfaz y navegación
  • Gestión de archivos y carpetas
  • Configuración de red
  • Comandos CMD y PowerShell
  • Gestión de usuarios y permisos

Fase 3: Linux (4-5 semanas)

  • Instalación y distribuciones
  • Línea de comandos básica
  • Estructura de directorios
  • Gestión de archivos y permisos
  • Configuración de red
  • Scripting básico (bash)

Fase 4: Administración Avanzada (3-4 semanas)

  • Gestión de procesos
  • Administración de memoria
  • Configuración de servicios
  • Monitorización y logging
  • Seguridad y firewall
  • Backup y recuperación

Fase 5: Práctica y Proyectos (4-6 semanas)

  • Configurar servidor web Apache/Nginx
  • Montar servidor de archivos
  • Configurar red doméstica
  • Automatizar tareas con scripts
  • Resolver problemas prácticos
  • Preparación para certificaciones
