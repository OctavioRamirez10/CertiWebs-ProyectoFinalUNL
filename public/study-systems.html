💡 Comandos Esenciales

Comandos Básicos Linux

# Navegación y archivos ls -la # Listar archivos detallados cd /home/user # Cambiar directorio pwd # Mostrar directorio actual mkdir nueva_carpeta # Crear directorio rm archivo.txt # Eliminar archivo cp origen destino # Copiar archivo mv origen destino # Mover/renombrar # Gestión de usuarios sudo useradd nuevo_user # Crear usuario sudo passwd nuevo_user # Cambiar contraseña sudo userdel nuevo_user # Eliminar usuario groups username # Mostrar grupos de usuario # Permisos chmod 755 archivo # Cambiar permisos chown user:group archivo # Cambiar propietario ls -l archivo # Ver permisos

Comandos Windows PowerShell

# Navegación y archivos Get-ChildItem # Listar archivos Set-Location C:\Users # Cambiar directorio New-Item -ItemType Directory carpeta # Crear directorio Remove-Item archivo.txt # Eliminar archivo Copy-Item origen destino # Copiar archivo Move-Item origen destino # Mover archivo # Gestión de usuarios Get-LocalUser # Listar usuarios New-LocalUser -Name "user" -PasswordNeverExpires Set-LocalUser -Name "user" -Password (Read-Host -AsSecureString) Remove-LocalUser -Name "user" # Servicios y procesos Get-Service # Listar servicios Start-Service serviceName # Iniciar servicio Stop-Service serviceName # Detener servicio Get-Process # Listar procesos Stop-Process -Name "process" # Matar proceso

Administración del Sistema

# Linux - Gestión de procesos ps aux # Ver procesos top # Monitor de procesos kill -9 PID # Matar proceso systemctl status nginx # Estado de servicio systemctl start nginx # Iniciar servicio systemctl enable nginx # Habilitar al inicio # Windows - Monitorización tasklist # Ver procesos taskkill /F /PID 1234 # Matar proceso Get-EventLog System # Ver logs perfmon # Monitor de rendimiento sfc /scannow # Verificar archivos sistema

Configuración de Red

# Linux ip addr show # Ver interfaces ip route show # Ver tabla rutas ping -c 4 google.com # Prueba conectividad ss -tuln # Ver puertos abiertos ufw status # Estado firewall # Windows ipconfig /all # Ver configuración IP netstat -an # Ver conexiones ping google.com # Prueba conectividad netsh advfirewall show allprofiles nslookup google.com # Resolución DNS
