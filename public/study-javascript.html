JavaScript - Guía de Estudio

Aprende programación web dinámica con JavaScript, el lenguaje más popular del mundo

¿Qué es JavaScript?

JavaScript es un lenguaje de programación interpretado que permite crear contenido web dinámico e interactivo. Es el lenguaje principal del lado del cliente en los navegadores web y también se puede usar en el servidor con Node.js.

Características Principales

  • Lenguaje interpretado: No necesita compilación
  • Dinámico: Tipos de datos flexibles
  • Multi-paradigma: Soporta POO, funcional y procedural
  • Event-driven: Responde a eventos del usuario
  • Asincrónico: Manejo de operaciones no bloqueantes
  • Universal: Corre en navegadores y servidores

Ejemplo de Código

// Variables y constantes
const nombre = "Juan";
let edad = 25;
var esEstudiante = true;

// Funciones
function saludar(nombre) {
  return `¡Hola, ${nombre}!`;
}

// Arrow function
const sumar = (a, b) => a + b;

// Objetos
const persona = {
  nombre: "Ana",
  edad: 30,
  saludar() {
    return `Hola, soy ${this.nombre}`;
  }
};

// Arrays
const numeros = [1, 2, 3, 4, 5];
const duplicados = numeros.map(n => n * 2);

// DOM manipulation
document.getElementById('boton').addEventListener('click', () => {
  alert('¡Botón clickeado!');
});

Conceptos Fundamentales

Variables y Tipos de Datos

  • var, let, const: Diferencias y uso
  • Tipos primitivos: string, number, boolean, null, undefined
  • Tipos complejos: object, array, function
  • Type coercion: Conversión automática de tipos
  • Template literals: Strings con interpolación

Funciones

  • Declaración de funciones: function nombre() {}
  • Arrow functions: () => {}
  • Parámetros y argumentos: Parámetros por defecto
  • Scope: Ámbito de variables
  • Closures: Funciones que recuerdan su entorno

Objetos y Arrays

  • Notación literal: {} y []
  • Propiedades y métodos: Acceso y modificación
  • Métodos de array: map, filter, reduce, forEach
  • Destructuring: Extracción de valores
  • Spread operator: ... para copiar y combinar

Control de Flujo

  • Condicionales: if/else, switch
  • Bucles: for, while, do-while
  • Iteración: for...in, for...of
  • Operadores ternarios: condición ? valor1 : valor2
  • Short-circuit: && y || para condicionales

JavaScript Moderno (ES6+)

ES6 y Características Modernas

  • let y const: Block scope y variables inmutables
  • Arrow functions: Sintaxis concisa y this léxico
  • Template literals: Strings con expresiones
  • Destructuring: Extracción de datos
  • Default parameters: Parámetros por defecto
  • Rest/Spread: Operadores ...
  • Classes: Sintaxis para POO
  • Promises: Manejo de asincronía
  • Async/await: Azúcar sintáctico para promises
  • Modules: import/export

Ejemplo de Código Moderno

// Async/Await
const obtenerDatos = async () => {
  try {
    const response = await fetch('/api/datos');
    const datos = await response.json();
    return datos;
  } catch (error) {
    console.error('Error:', error);
  }
};

// Classes
class Usuario {
  constructor(nombre, email) {
    this.nombre = nombre;
    this.email = email;
  }
  
  saludar() {
    return `Hola, soy ${this.nombre}`;
  }
}

// Destructuring
const { nombre, email } = usuario;
const [primero, segundo] = array;

// Spread operator
const nuevoArray = [...array1, ...array2];
const nuevoObjeto = { ...obj1, ...obj2 };

DOM Manipulación

Selección de Elementos

  • getElementById: Selección por ID
  • querySelector/querySelectorAll: Selectores CSS
  • getElementsByClassName/TagName: Selección múltiple
  • Traversal: Navegación por el DOM

Manipulación de Elementos

  • textContent/innerHTML: Modificar contenido
  • style: Modificar estilos CSS
  • classList: Añadir/eliminar clases
  • setAttribute/getAttribute: Atributos
  • createElement/appendChild: Crear elementos

Eventos

  • addEventListener: Manejo de eventos
  • Event types: click, submit, keydown, mouseover
  • Event object: Información del evento
  • Event propagation: Bubbling y capturing
  • Event delegation: Delegación de eventos

Plan de Estudio Recomendado

Fase 1: Fundamentos (2-3 semanas)

  • Sintaxis básica y estructura
  • Variables y tipos de datos
  • Operadores y expresiones
  • Control de flujo
  • Funciones básicas

Fase 2: Estructuras de Datos (2-3 semanas)

  • Arrays y métodos de array
  • Objetos y propiedades
  • Manipulación de strings
  • JSON y parseo de datos
  • Algoritmos básicos

Fase 3: DOM y Eventos (2-3 semanas)

  • Selección de elementos
  • Manipulación del DOM
  • Manejo de eventos
  • Formularios y validación
  • Animaciones básicas

Fase 4: JavaScript Moderno (3-4 semanas)

  • ES6+ features
  • Promises y async/await
  • Fetch API y AJAX
  • Modules y organización de código
  • Debugging y herramientas

Fase 5: Proyectos Prácticos (4-6 semanas)

  • Calculadora interactiva
  • Todo list app
  • Weather app con API
  • Juego simple en browser
  • Portfolio personal interactivo
