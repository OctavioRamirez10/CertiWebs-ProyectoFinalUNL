💡 Ejemplos Prácticos

Variables y Tipos

// Variables modernas let nombre = "Juan"; const edad = 25; var ciudad = "Madrid"; // Tipos de datos let texto = "Hola Mundo"; let numero = 42; let booleano = true; let array = [1, 2, 3]; let objeto = { nombre: "Juan", edad: 25 };

Funciones Modernas

// Función tradicional function sumar(a, b) { return a + b; } // Función flecha const multiplicar = (a, b) => a * b; // Función asíncrona async function obtenerDatos() { try { const response = await fetch('/api/datos'); const datos = await response.json(); return datos; } catch (error) { console.error('Error:', error); } }

Manipulación del DOM

// Seleccionar elementos const boton = document.getElementById('miBoton'); const parrafo = document.querySelector('.mi-parrafo'); // Event listeners boton.addEventListener('click', () => { parrafo.textContent = '¡Texto cambiado!'; parrafo.style.color = '#ff6b35'; }); // Crear elementos const nuevoDiv = document.createElement('div'); nuevoDiv.className = 'nuevo-elemento'; nuevoDiv.textContent = 'Elemento dinámico'; document.body.appendChild(nuevoDiv);

Arrays y Métodos

const numeros = [1, 2, 3, 4, 5]; // Métodos funcionales const duplicados = numeros.map(n => n * 2); const pares = numeros.filter(n => n % 2 === 0); const suma = numeros.reduce((acc, n) => acc + n, 0); // Destructuring const [primero, segundo] = numeros; const { nombre, edad } = { nombre: "Ana", edad: 30 };
