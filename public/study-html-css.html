HTML5 Semántico

HTML5 introdujo etiquetas semánticas que describen el significado del contenido, mejorando accesibilidad y SEO.

  • <header> — Cabecera del sitio o sección.
  • <nav> — Bloque de navegación principal.
  • <main> — Contenido principal único de la página.
  • <article> — Contenido independiente y reutilizable.
  • <section> — Agrupación temática de contenido.
  • <aside> — Contenido lateral o complementario.
  • <footer> — Pie de página.
  • <figure> / <figcaption> — Imágenes con descripción.
<!-- Estructura semántica básica --> <header> <nav><a href="/">Inicio</a></nav> </header> <main> <article> <h1>Título principal</h1> <section>Sección de contenido</section> </article> </main> <footer>© 2025 CertiWebs</footer>
💡 Tip: Usar etiquetas semánticas correctas mejora la puntuación en herramientas de accesibilidad como Lighthouse.

Formularios HTML5

Los formularios HTML5 incluyen validación nativa, nuevos tipos de input y atributos mejorados.

  • type="email" — Valida formato de correo automáticamente.
  • type="number" — Solo acepta valores numéricos.
  • type="date" — Selector de fecha nativo.
  • required — Campo obligatorio.
  • pattern — Expresión regular de validación.
  • placeholder — Texto de ayuda dentro del campo.
  • min / max — Rango permitido para números o fechas.
<form action="/enviar" method="POST"> <label>Email:</label> <input type="email" name="email" required> <label>Edad (18-99):</label> <input type="number" min="18" max="99" required> <button type="submit">Enviar</button> </form>

CSS Flexbox

Flexbox permite distribuir elementos en una sola dimensión (fila o columna) con control preciso del alineamiento.

  • display: flex — Activa el contexto flex en el contenedor.
  • flex-direction — row | column | row-reverse | column-reverse.
  • justify-content — Alineación en eje principal: flex-start, center, space-between, space-around.
  • align-items — Alineación en eje secundario: stretch, center, flex-start, flex-end.
  • gap — Espacio entre elementos.
  • flex-wrap: wrap — Los elementos saltan de línea si no caben.
  • flex: 1 — El elemento ocupa el espacio disponible restante.
/* Navbar con flexbox */ .navbar { display: flex; justify-content: space-between; align-items: center; gap: 1rem; padding: 1rem 2rem; } /* Centrar un elemento */ .centrado { display: flex; justify-content: center; align-items: center; min-height: 100vh; }

CSS Grid Layout

CSS Grid es ideal para layouts bidimensionales (filas y columnas simultáneas).

  • display: grid — Activa el layout grid.
  • grid-template-columns — Define el ancho de cada columna.
  • grid-template-rows — Define la altura de cada fila.
  • repeat(n, valor) — Repite una definición n veces.
  • 1fr — Fracción del espacio disponible.
  • minmax(min, max) — Columna con tamaño mínimo y máximo.
  • grid-column / grid-row — Controla cuántas celdas ocupa un elemento.
  • gap — Espacio entre filas y columnas.
/* Grid responsivo de 3 columnas */ .grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(280px, 1fr)); gap: 2rem; } /* Elemento que ocupa 2 columnas */ .featured { grid-column: span 2; }

Responsive Design & Media Queries

El diseño responsivo adapta la interfaz a cualquier tamaño de pantalla usando media queries.

  • Breakpoints comunes: 480px (móvil), 768px (tablet), 1024px (desktop).
  • mobile-first: Se diseña primero para móvil y se expande con min-width.
  • viewport meta: <meta name="viewport" content="width=device-width, initial-scale=1">
  • Unidades relativas: %, vw, vh, rem, em en lugar de píxeles fijos.
/* Mobile first */ .columnas { display: flex; flex-direction: column; gap: 1rem; } /* Tablet y arriba */ @media (min-width: 768px) { .columnas { flex-direction: row; flex-wrap: wrap; } } /* Desktop */ @media (min-width: 1024px) { .columnas { display: grid; grid-template-columns: repeat(3, 1fr); } }

Variables CSS y Animaciones

Las Custom Properties (variables CSS) y las animaciones elevan la calidad y mantenibilidad del CSS.

  • --variable: valor — Declarada dentro de :root para alcance global.
  • var(--variable) — Se usa para leer el valor.
  • transition — Animación suave entre estados (hover, focus).
  • @keyframes — Define animaciones personalizada paso a paso.
  • animation — Aplica un @keyframes a un elemento.
  • transform — Rotación, escala, traslación sin afectar el flujo.
/* Variables globales */ :root { --color-primario: #00f3ff; --radio: 12px; } .boton { background: var(--color-primario); border-radius: var(--radio); transition: transform 0.3s ease, box-shadow 0.3s ease; } .boton:hover { transform: translateY(-4px); box-shadow: 0 8px 20px rgba(0,243,255,0.4); } @keyframes fadeIn { from { opacity: 0; transform: translateY(20px); } to { opacity: 1; transform: translateY(0); } } .tarjeta { animation: fadeIn 0.5s ease; }
