HTML & CSS - Guía de Estudio

Aprende los fundamentos del desarrollo web frontend con HTML y CSS

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Define la estructura y el contenido de los documentos web mediante etiquetas y elementos.

Conceptos Fundamentales

  • Estructura básica: DOCTYPE, html, head, body
  • Etiquetas semánticas: header, nav, main, section, article, footer
  • Texto y contenido: h1-h6, p, span, div
  • Enlaces y medios: a, img, video, audio
  • Formularios: form, input, textarea, select, button
  • Listas: ul, ol, li, dl, dt, dd
  • Tablas: table, thead, tbody, tr, th, td

Ejemplo de Código

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Mi Página</title>
</head>
<body>
  <header>
    <h1>Bienvenidos</h1>
  </header>
  <main>
    <p>Contenido principal</p>
  </main>
</body>
</html>

¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje utilizado para describir la presentación y el estilo de los documentos HTML. Controla el diseño, colores, fuentes y espaciado.

Conceptos Fundamentales

  • Selectores: por etiqueta, clase, ID, atributo
  • Modelo de caja: margin, border, padding, content
  • Display y posicionamiento: block, inline, flexbox, grid
  • Colores y fondos: color, background, gradients
  • Tipografía: font-family, font-size, font-weight
  • Responsive Design: media queries, unidades relativas
  • Animaciones: transitions, animations, transforms

Ejemplo de Código

.container {
  display: flex;
  justify-content: center;
  align-items: center;
  padding: 2rem;
}

.button {
  background: #ff6b35;
  color: white;
  padding: 1rem 2rem;
  border: none;
  border-radius: 8px;
  cursor: pointer;
  transition: all 0.3s ease;
}

.button:hover {
  background: #e55a2b;
  transform: translateY(-2px);
}

Plan de Estudio Recomendado

Fase 1: Fundamentos HTML (1-2 semanas)

  • Estructura básica y sintaxis
  • Etiquetas de texto y semánticas
  • Enlaces e imágenes
  • Listas y tablas
  • Formularios básicos

Fase 2: Fundamentos CSS (2-3 semanas)

  • Selectores y sintaxis
  • Modelo de caja
  • Tipografía y colores
  • Posicionamiento y flotación
  • Responsive Design básico

Fase 3: Layout Moderno (2-3 semanas)

  • Flexbox
  • Grid Layout
  • Media Queries avanzadas
  • Animaciones y transiciones
  • Variables CSS

Fase 4: Proyectos Prácticos (3-4 semanas)

  • Clonar una página web simple
  • Crear un portfolio personal
  • Diseñar una landing page
  • Construir una aplicación pequeña
  • Optimizar para móviles

Rendir Examen de HTML & CSS

Preparación para el Examen

Temas del Examen

  • Estructura HTML5: DOCTYPE, head, body, etiquetas semánticas
  • Formularios HTML: input types, validación, atributos
  • Medios y Multimedia: img, video, audio, responsive images
  • Selectores CSS: por clase, ID, atributo, pseudo-clases
  • Modelo de Caja: margin, padding, border, box-sizing
  • Layout CSS: display, position, flexbox, grid
  • Responsive Design: media queries, unidades relativas
  • Tipografía y Colores: font-family, font-size, color, background

Ejercicios Prácticos

  • Ejercicio 1: Crear una página web con estructura semántica HTML5
  • Ejercicio 2: Diseñar un formulario con validación CSS
  • Ejercicio 3: Implementar un layout responsive con Flexbox
  • Ejercicio 4: Crear animaciones CSS simples
  • Ejercicio 5: Optimizar imágenes para diferentes dispositivos

Consejos para el Examen

  • Practica escribiendo código a mano sin autocompletado
  • Memoriza las propiedades CSS más comunes
  • Entiende la diferencia entre block, inline e inline-block
  • Aprende a depurar CSS usando las herramientas del navegador
  • Practica con proyectos reales pequeños
  • Revisa las especificaciones oficiales de W3C

Simulador de Examen


¿Cuál es la etiqueta correcta para crear un formulario?

A) <form>
B) <input>
C) <table>
D) <div>

Respuesta: A) <form>


La etiqueta <form> se utiliza para crear formularios web
que permiten a los usuarios enviar datos al servidor.
